# 一些关于包装类应用的题目

1. 判断一下obj1和obj2的值是否相同  

```java
    @Test
        public void packApp() {
            Object o1=true?new Integer(1):new Double(2.0);
            Object o2;
            if(true) {
                o2=new Integer(1);
            }else {
                o2=new Double(2.0);
            }
            System.out.println(o1);//1.0
            System.out.println(o2);//1
        }
```
